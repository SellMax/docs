Productos

El elemento principal del sistema son los productos y su manejo de inventario, saber dar de alta, actualizarlos, ver su información y removerlos es lo más necesario.

Aspectos generales

En la tabla nos encontraremos diferentes elementos.

  • Una lista desplegable donde podrá elegir cuantos productos mostrar al mismo tiempo.
  • Una barra de busqueda para buscar algún producto, ya sea por nombre, SKU o código único.
  • La información del producto en el siguiente orden:
    1. Nombre del producto (y su imágen).
    2. Código de barras o SKU.
    3. Código único (utilizado para los pedidos a Proveedores).
    4. Precio de venta.
    5. Cantidad disponible en stock.
    6. El estado actual del producto (ver más).
    7. Las diferentes acciones que puedes realizar con el producto.

Estados del producto

Puede toparse con diferentes estados del producto, pero no se preocupe, a continuación se los explicamos:

  • Disponible. Tiene buen stock y puede agregarlo a ventas.
  • Por agotarse. Al momento de crear el producto, usted definió el mínimo de stock del producto, cuando el stock actual del mismo este por debajo de dicha cantidad, aparecerá este estado para avisarle que está llegando al límite, pero puede seguir agregandolo a ventas.
  • Agotado. El stock del producto llegó a 0 y no es posible seguir agregandolo a ventas.

Acciones/eventos

En el apartado de Acción, tenemos 3 botones que nos ayudan a lanzar eventos para cada producto.

  • Ver detalles producto.
  • Editar/Actualizar producto.
  • Remover producto.

Crear nuevo producto

El primer paso para comenzar a vender es dar sus productos de alta, pero no se preocupe, depende el giro de su negocio, es posible que se le ofrecieran bases de datos precargadas, es decir, que ya tenemos miles de productos listos para que usted coloque algunos campos y listo. Igual no se asuste, le explicamos paso a paso.

Empecemos por el principio, para crear un nuevo producto basta con dar click en el boton "Crear producto" y le redigirá a la página para hacerlo.

Una vez en la página, tenemos varios campos, algunos estan marcados con un asterisco (*), eso significa que son campos obligatorios, es decir, que forzosamente deben contener información, esos campos son:

  • Producto. El nombre del producto.
  • Unidad. ¿Cómo se vende el producto?, por pieza, por metro, litro, etc., el sistema tiene por defecto una sola opción (Pieza), pero usted puede agregar más desde unidades.
  • Cantidad. Aquí debe ingresar el stock inicial del producto, cuantos tiene actualmente.
  • Cantidad mínima. Cuantos productos debe haber mínimo, para que el sistema detecte que está por agotarse.
  • Cantidad máxima. Cuantos productos máximos debe haber, esto para el momento de hacer los reportes.
  • Código de barras. Aquí debe ingresar el código de barras o SKU del producto, es con el que podrá buscar y agregar el producto en las ventas.
    NOTA - Deben ser códigos únicos y no pueden repetirse.
  • Código único. Este código es el que su proveedor le da para poder resurtirle el producto, puede ser igual al SKU o diferente.
    NOTA - Deben ser códigos únicos y no pueden repetirse.
  • Finanzas. Esta es la categoría para que usted lleve un control de sus finanzas más explicito y riguroso; con estas categorías puede separar el dinero que ingresa, por ejemplo, lo que tiene referente a frutas, licores, etc., el sistema por defecto solo cuenta con la categoría General, pero usted puede agregar más desde categorías de finanzas.
  • Precio compra. Aquí debe ingresar el precio por el cual adquirió el producto.
  • Precio venta. Este es el precio de venta al público.

Ademas de los obligatorios, hay campos que son opcionales, pero que le ayudarán al mejor manejo y control de sus reportes.

  • Categoría. A qué departamento o categoría pertenece el producto.
  • Subcategoría. La subcategoría referente a la categoría previamente seleccionada.
  • Marca. Qué marca es el producto.
  • Proveedor. Que proveedor le surte dicho producto; este para realizar ordenes de pedido por proveedor.
  • Pedido. Si el producto se incluirá en la orden del pedido o no.
  • Impuestos. Si el precio de venta incluye o no impuestos.
  • Ganancia. Cuanto porcentaje de ganancia le quiere sacar al producto.
    NOTA - Ingresar valores en los campos precio compra, impuestos y ganacia, afectará al precio de venta.
  • Imagen. Puede agregarle una imágen personalizada al producto.

Productos precargados

Como ya lo mencionamos anteriormente, contamos con bases de datos con productos precargados; actualmente solo tenemos para los siguientes giros:

  1. Abarrotes (+9,000 productos).
  2. Refaccionaria de motos (+1,500 productos).

Si su sistema incluía alguna de estas bases, es muy sencillo comenzar a utilizar y registrar los productos precargados.

NOTA - La única información que se cargará será el nombre del producto y su SKU; usted debera modificar todos los otros campos.

Para comenzar, debemos dar click en el botón "Cargar producto", dentro de la página "Crear producto".

Posteriormente se abrirá la ventana para escanear el código, ahí deberá escanear el código del producto que desea registrar, y si está registrado en la base de datos precargada, se agregará la información a los campos correspondientes.

De ahí, ya usted deberá terminar de configurar el producto.

Actualizar producto

La página para actualizar el producto es identica a la de crear producto, solo que va cargada con la información del producto el cual se va a actualizar.

Las instrucciones son las mismas que en crear producto.

Información de producto

Aquí podemos ver toda la información del producto en forma ordenada e imprimir un reporte completo del mismo.

Tambien podremos imprimir el código de barras del producto.

Remover producto

En nuestro sistema NO se pueden eliminar productos por completo, solo se inhabilitan. Esto lo hacemos con la finalidad de que no haya lagunas o datos perdidos en reportes.

Para remover un producto, solo debe ir a la página principal de productos, buscar el producto que desea remover y en el menú de Acción, dar click sobre el icono .

Posteriormente saldrá una ventana para confirmar la acción, debe aceptar y listo.

Productos removidos

Como ya se mencinó arriba, los productos solo se inhabilitan, por lo tanto, puedes volver a habilitarlos; para ver los productos inhabilitados y volverlos a activar, puede verlos dando click en el boton "Eliminados".

Aquí encontrará una tabla con todos los productos removidos, con la siguiente información:

  1. #. Es el identificador o numero de producto.
  2. Producto. El nombre del producto.
  3. Código. El código o SKU del producto.
  4. C. Único. El código único del producto.
  5. Precio ($). Precio del producto cuando fue eliminado.
  6. Existencia. La existencia del producto previo a que fuera removido.

Restaurar producto

Para restaurar un producto inhabilitado, solo debe dirigirse al producto que desea restaurar y dar click cobre el icono .

Al dar click, saldra una ventana de confirmación y solo debe aceptar.

NOTA - Al restarurar un producto, lo hará con 0 stock, por lo que deberá ajustar su inventario.
